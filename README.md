# Kaur Complex - Luxury Real Estate Website

A modern, responsive website for Kaur Complex, a premium real estate development in Peshawar featuring 600+ international standard apartments, shops, and luxury penthouses in a gated community.

## 🌟 Features

### Design & User Experience
- **Modern & Elegant UI** with white, gold, and navy blue color scheme
- **Fully Responsive** design optimized for all devices
- **Smooth Animations** and interactive elements
- **Mobile-First** approach
- **Sticky Navigation** with smooth scrolling
- **Loading Animations** and page transitions

### Pages & Content
1. **Home Page** - Hero section, quick stats, features, gallery preview
2. **About Us** - Project overview, mission, vision, developer background
3. **Apartments** - Detailed listings for 1-bed, 2-bed, 3-bed, and penthouses
4. **Shops & Commercial** - Commercial spaces with investment details
5. **Amenities** - Complete facilities showcase
6. **Gallery** - 3D renders, real images, progress updates
7. **Floor Plans** - Interactive layouts and downloadable PDFs
8. **Contact** - Contact form, map, office information

### Interactive Features
- **WhatsApp Integration** - Floating chat button
- **Contact Forms** - Inquiry and booking forms
- **Image Gallery** - Lightbox with filtering
- **Progress Timeline** - Construction updates
- **Counter Animations** - Animated statistics
- **Smooth Scrolling** - Enhanced navigation
- **Back to Top** - User-friendly navigation

### Technical Features
- **SEO Optimized** - Meta tags, structured data
- **Fast Loading** - Optimized images and code
- **Cross-Browser Compatible** - Works on all modern browsers
- **Accessibility** - WCAG compliant design
- **Print Styles** - Optimized for printing

## 🎨 Design System

### Color Palette
- **Primary**: Navy Blue (#1e3a8a)
- **Secondary**: Gold (#d4af37)
- **Accent**: Light Gold (#fbbf24)
- **White**: (#ffffff)
- **Light Gray**: (#f8fafc)
- **Dark Gray**: (#1f2937)

### Typography
- **Headings**: Playfair Display (serif)
- **Body**: Poppins (sans-serif)
- **Icons**: Font Awesome 6.4.0

### Components
- **Buttons**: Primary, Secondary, and CTA styles
- **Cards**: Feature cards, apartment cards, amenity cards
- **Forms**: Contact forms with validation
- **Navigation**: Sticky navbar with mobile menu
- **Gallery**: Image grid with lightbox
- **Timeline**: Progress tracking

### File Setup
1. Replace placeholder images in `assets/images/` with actual project images
2. Update contact information in all HTML files
3. Replace Google Maps embed URL with actual location
4. Update WhatsApp number in all pages
5. Add actual PDF documents to `assets/documents/`

## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px
- **Large Desktop**: > 1400px

## 🔧 Customization

### Adding New Pages
1. Create new HTML file in `pages/` directory
2. Copy navigation structure from existing pages
3. Update navigation links in all pages
4. Add page-specific styles to CSS files

### Modifying Colors
1. Update CSS custom properties in `assets/css/style.css`
2. Colors are defined in `:root` selector
3. All components use these variables for consistency

### Adding Content
1. **Images**: Add to appropriate folder in `assets/images/`
2. **Documents**: Add PDFs to `assets/documents/`
3. **Videos**: Add to `assets/videos/` folder
